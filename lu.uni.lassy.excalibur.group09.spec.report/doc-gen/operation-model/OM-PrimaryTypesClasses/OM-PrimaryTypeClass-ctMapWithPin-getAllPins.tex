\subsection{Operation Model for getAllPins}

\label{OM-getAllPins}


The \msrcode{getAllPins} operation has the following properties:

	\begin{operationmodel}
	\addheading{Operation}
	\adddoublerow{getAllPins}{used to get the geographical positions of the pins that is currently shown on the map. (usually used only when there's actually a single pin on the map)
	 }


	\addrowheading{Return type}
	\addsinglerow{ptBoolean}

		

	\addrowheading{Post-Condition (functional)}
	\addnumberedsinglerow{PostF}{returns a list of geographical positions.}

	\end{operationmodel}
