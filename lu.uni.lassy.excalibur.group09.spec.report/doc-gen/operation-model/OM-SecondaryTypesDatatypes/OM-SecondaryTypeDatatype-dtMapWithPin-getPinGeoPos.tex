\subsection{Operation Model for getPinGeoPos}

\label{OM-getPinGeoPos}


The \msrcode{getPinGeoPos} operation has the following properties:

	\begin{operationmodel}
	\addheading{Operation}
	\adddoublerow{getPinGeoPos}{used to get the geographical position of the pin that is currently shown on the map (used only when there's a single pin on the map).
	 }


	\addrowheading{Return type}
	\addsinglerow{dtGeoPos}

		

	\addrowheading{Post-Condition (functional)}
	\addnumberedsinglerow{PostF}{Returns a geographical position of the pin that was on the map.}

	\end{operationmodel}
